## Welcome to Publive

Publive is an India-first headless CMS built for modern digital publishers and enterprises. Get up and running in three steps:

<Steps>
  <Step title="Authenticate" href="/documentation/getting-started/authentication">
    Generate your API key and set up credentials for the CDS or CMS API.
  </Step>
  <Step title="Fetch Content" href="/documentation/getting-started/quick-start">
    Make your first API call to retrieve posts, categories, or site config.
  </Step>
  <Step title="Build Your Frontend" href="/documentation/guides/frontend-integration">
    Integrate with React, Next.js, Angular, Vue, or any framework of your choice.
  </Step>
</Steps>

<CardGroup cols={2}>
  <Card
    title="Content Delivery API (CDS)"

  </Card>
  <Card
    title="Content Management API (CMS)"
    icon="pen-to-square"

  </Card>
</CardGroup>
