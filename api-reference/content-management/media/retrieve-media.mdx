---
title: "Retrieve Media"
description: Retrieve a single media asset by ID
api: "GET https://cms.thepublive.com/publisher/{publisher_id}/media-library/{id}/"
---

Returns the details of a specific media asset.

<ParamField path="publisher_id" type="string" required>
  Your Publisher ID
</ParamField>

<ParamField path="id" type="integer" required>
  Media ID
</ParamField>

<ParamField header="username" type="string" required>
  API Key
</ParamField>

<ParamField header="password" type="string" required>
  API Secret
</ParamField>
