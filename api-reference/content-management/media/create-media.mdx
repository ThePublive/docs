---
title: "Create Media"
description: Create/upload a media asset
api: "POST https://cms.thepublive.com/publisher/{publisher_id}/media-library/"
---

Creates a new media entry in the library.

<ParamField path="publisher_id" type="string" required>
  Your Publisher ID
</ParamField>

<ParamField header="username" type="string" required>
  API Key
</ParamField>

<ParamField header="password" type="string" required>
  API Secret
</ParamField>

<ParamField body="filename" type="string" required>
  Filename
</ParamField>

<ParamField body="path" type="string" required>
  Media URL
</ParamField>

<ParamField body="alt_text" type="string">
  Alt text for accessibility
</ParamField>

<ParamField body="caption" type="string">
  Caption/description
</ParamField>

<ParamField body="source" type="string">
  Source/credit
</ParamField>

<ParamField body="type" type="string">
  `Image`, `Video`, or `File`
</ParamField>

<ParamField body="meta_data" type="object">
  Metadata (width, height, etc.)
</ParamField>

#### Example Request

```bash
curl -X POST \
  'https://cms.thepublive.com/publisher/123/media-library/' \
  -H 'username: YOUR_API_KEY' \
  -H 'password: YOUR_API_SECRET' \
  -H 'Content-Type: application/json' \
  -d '{
    "filename": "hero-image.jpg",
    "path": "https://cdn.publive.com/images/hero-image.jpg",
    "alt_text": "Hero banner image",
    "source": "Staff Photographer"
  }'
```
