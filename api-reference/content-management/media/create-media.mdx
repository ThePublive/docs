---
title: "Register Media"
description: Register a media file or an existing media URL to the CMS library
api: "POST https://cms.thepublive.com/publisher/{publisher_id}/media-library/"
---

Registers media (file or existing url) to the CMS library. 

Creates a new media entry in the library, either by uploading a file directly (`multipart/form-data`) or by referencing an external URL (`application/json`).

<ParamField path="publisher_id" type="string" required>
  Your Publisher ID
</ParamField>



<ParamField body="filename" type="string" required>
  Filename
</ParamField>

<Note>
  Provide **either** `path` **or** `media` — exactly one is required, not both.
</Note>

<ParamField body="path" type="string">
  <Badge color="orange" size="sm">Immutable</Badge> Media URL. Required if `media` is not provided.
</ParamField>

<ParamField body="media" type="file">
  <Badge color="orange" size="sm">Immutable</Badge> Media file. Required if `path` is not provided.
</ParamField>

<ParamField body="alt_text" type="string">
  Alt text for accessibility
</ParamField>

<ParamField body="caption" type="string">
  Caption/description
</ParamField>

<ParamField body="source" type="string">
  Source/credit
</ParamField>

<ParamField body="type" type="string">
  <Badge color="orange" size="sm">Immutable</Badge> `Image`, `Video`, or `File`.
</ParamField>

<ParamField body="meta_data" type="object">
  Metadata (width, height, etc.)
</ParamField>

#### Example Request

<CodeGroup>

```bash File Upload (multipart/form-data)
curl -X POST \
  'https://cms.thepublive.com/publisher/123/media-library/' \
  -H 'Authorization: Basic <BASE64_AUTH_TOKEN>' \
  -F 'filename=hero-image.jpg' \
  -F 'media=@/path/to/hero-image.jpg' \
  -F 'alt_text=Hero banner image' \
  -F 'source=Staff Photographer'
```

```bash External URL (application/json)
curl -X POST \
  'https://cms.thepublive.com/publisher/123/media-library/' \
  -H 'Authorization: Basic <BASE64_AUTH_TOKEN>' \
  -H 'Content-Type: application/json' \
  -d '{
    "filename": "hero-image.jpg",
    "path": "https://cdn.publive.com/images/hero-image.jpg",
    "alt_text": "Hero banner image",
    "source": "Staff Photographer"
  }'
```

</CodeGroup>
