---
title: "Create Post"
description: Create a new post
api: "POST https://cms.thepublive.com/publisher/{publisher_id}/post/"
---

Creates a new post. Posts are created with `Draft` status by default.

<ParamField path="publisher_id" type="string" required>
  Your Publisher ID
</ParamField>

<ParamField header="username" type="string" required>
  API Key
</ParamField>

<ParamField header="password" type="string" required>
  API Secret
</ParamField>

<ParamField body="title" type="string" required>
  Post headline
</ParamField>

<ParamField body="english_title" type="string" required>
  English headline (for slug)
</ParamField>

<ParamField body="type" type="string" required>
  `Article`, `Video`, `Web Story`, `Gallery`, `LiveBlog`, `CustomPage`, `BlankPage`
</ParamField>

<ParamField body="status" type="string" required>
  `Draft`, `Published`, `Scheduled`, `Approval Pending`
</ParamField>

<ParamField body="primary_category" type="integer" required>
  Category ID
</ParamField>

<ParamField body="contributors" type="string">
  Comma-separated author IDs
</ParamField>

<ParamField body="content" type="string">
  HTML body content
</ParamField>

<ParamField body="tags" type="string">
  Comma-separated tag IDs
</ParamField>

<ParamField body="categories" type="string">
  Comma-separated additional category IDs
</ParamField>

<ParamField body="banner_url" type="integer">
  Media ID for featured image
</ParamField>

<ParamField body="banner_description" type="string">
  Featured image caption
</ParamField>

<ParamField body="short_description" type="string">
  SEO meta description
</ParamField>

<ParamField body="summary" type="string">
  Post summary
</ParamField>

<ParamField body="seo_keyphrase" type="string">
  Focus keyword
</ParamField>

<ParamField body="slug" type="string">
  Custom slug (auto-generated if omitted)
</ParamField>

<ParamField body="meta_data" type="object">
  Additional metadata (e.g., `{"access_type": "Paid"}`)
</ParamField>

<ParamField body="custom_published_at" type="string">
  Backdated publish date
</ParamField>

<ParamField body="scheduled_at" type="string">
  Future publish date (status must be `Scheduled`)
</ParamField>

<ParamField body="hide_banner_image" type="boolean">
  Hide the featured image
</ParamField>

#### Example Request

```bash
curl -X POST \
  'https://cms.thepublive.com/publisher/123/post/' \
  -H 'username: YOUR_API_KEY' \
  -H 'password: YOUR_API_SECRET' \
  -H 'Content-Type: application/json' \
  -d '{
    "title": "Getting Started with Publive API",
    "english_title": "Getting Started with Publive API",
    "type": "Article",
    "status": "Draft",
    "primary_category": 100,
    "contributors": "1,2",
    "tags": "500,501",
    "content": "<p>Learn how to integrate Publive into your application...</p>",
    "short_description": "A beginner guide to the Publive API"
  }'
```
