---
title: "Delete Post"
description: Delete a post
api: "DELETE https://cms.thepublive.com/publisher/{publisher_id}/post/{id}/"
---

Permanently deletes a post.

<Warning>
**Warning:** This action cannot be undone. The post and all its associated data will be permanently removed.
</Warning>

<ParamField path="publisher_id" type="string" required>
  Your Publisher ID
</ParamField>

<ParamField path="id" type="integer" required>
  Post ID
</ParamField>

<ParamField header="username" type="string" required>
  API Key
</ParamField>

<ParamField header="password" type="string" required>
  API Secret
</ParamField>
