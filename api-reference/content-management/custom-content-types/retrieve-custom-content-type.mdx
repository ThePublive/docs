---
title: "Retrieve Custom Content Type"
description: Retrieve a single custom content type schema by ID
api: "GET https://cms.thepublive.com/publisher/{publisher_id}/entities/content-type/{custom_entity_schema_id}"
---

Returns the details of a specific custom content type schema.

<ParamField path="publisher_id" type="string" required>
  Your Publisher ID
</ParamField>

<ParamField path="custom_entity_schema_id" type="string" required>
  Custom content type schema ID
</ParamField>
