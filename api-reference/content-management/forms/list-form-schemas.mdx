---
title: "List Form Schemas"
description: Fetch all form schemas
api: "GET https://cms.thepublive.com/publisher/{publisher_id}/form/schema/"
---

Returns a list of all form schemas for the publisher.

<ParamField path="publisher_id" type="string" required>
  Your Publisher ID
</ParamField>

<ParamField header="Authorization" type="string" required>
  `Basic <BASE64_AUTH_TOKEN>`
</ParamField>

#### Example Request

```bash
curl -X GET \
  'https://cms.thepublive.com/publisher/123/form/schema/' \
  -H 'Authorization: Basic <BASE64_AUTH_TOKEN>'
```
