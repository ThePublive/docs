---
title: "Delete Category"
description: Delete a category
api: "DELETE https://cms.thepublive.com/publisher/{publisher_id}/category/{id}/"
---

Permanently deletes a category.

<Warning>
**Warning:** This action cannot be undone. Posts assigned to this category will need to be reassigned.
</Warning>

<ParamField path="publisher_id" type="string" required>
  Your Publisher ID
</ParamField>

<ParamField path="id" type="integer" required>
  Category ID
</ParamField>

<ParamField header="username" type="string" required>
  API Key
</ParamField>

<ParamField header="password" type="string" required>
  API Secret
</ParamField>
