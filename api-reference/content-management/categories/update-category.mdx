---
title: "Update Category"
description: Update an existing category
api: "PATCH https://cms.thepublive.com/publisher/{publisher_id}/category/{id}/"
---

Updates an existing category. Only send the fields you want to change (partial update).

<Warning>
The following fields are **immutable after creation** and cannot be changed via update:
- `english_name` — English name used for permalink generation
- `slug` — URL slug (derived from `english_name` at creation)
- `content_type` — Content type filter
</Warning>

#### Updatable fields

<ParamField path="publisher_id" type="string" required>
  Your Publisher ID
</ParamField>

<ParamField path="id" type="integer" required>
  Category ID
</ParamField>



<ParamField body="name" type="string">
  Category name
</ParamField>

<ParamField body="meta_title" type="string">
  SEO title
</ParamField>

<ParamField body="meta_description" type="string">
  SEO description
</ParamField>

<ParamField body="content" type="string">
  Category description (HTML)
</ParamField>

<ParamField body="category_brand_color" type="string">
  Brand color (hex)
</ParamField>

<ParamField body="priority" type="integer">
  Priority level
</ParamField>

#### Example request

```bash
curl -X PATCH \
  'https://cms.thepublive.com/publisher/123/category/14428/' \
  -H 'Authorization: Basic YOUR_BASE64_TOKEN' \
  -H 'Content-Type: application/json' \
  -d '{"name": "Latest News", "meta_title": "Latest News Updates"}'
```
