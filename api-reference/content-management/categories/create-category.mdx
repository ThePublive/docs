title: "Create Category"
description: Create a new category
api: "POST https://cms.thepublive.com/publisher/{publisher_id}/category/"
---

Creates a new category.

<ParamField path="publisher_id" type="string" required>
  Your Publisher ID
</ParamField>

<ParamField header="username" type="string" required>
  API Key
</ParamField>

<ParamField header="password" type="string" required>
  API Secret
</ParamField>

<ParamField body="name" type="string" required>
  Category name
</ParamField>

<ParamField body="english_name" type="string" required>
  English name (for permalink generation)
</ParamField>

<ParamField body="slug" type="string">
  Custom slug (auto-generated if omitted)
</ParamField>

<ParamField body="meta_title" type="string">
  SEO title
</ParamField>

<ParamField body="h1_tag" type="string">
  H1 heading tag
</ParamField>

<ParamField body="meta_description" type="string">
  SEO description
</ParamField>

<ParamField body="priority" type="integer">
  Priority level (1-1000)
</ParamField>

<ParamField body="content" type="string">
  Category description (HTML)
</ParamField>

<ParamField body="category_brand_color" type="string">
  Brand color (hex)
</ParamField>

<ParamField body="content_type" type="string">
  Content type filter (e.g., `Article`, `Web Story`)
</ParamField>

#### Example Request

```bash
curl -X POST \
  'https://cms.thepublive.com/publisher/123/category/' \
  -H 'username: YOUR_API_KEY' \
  -H 'password: YOUR_API_SECRET' \
  -H 'Content-Type: application/json' \
  -d '{
    "name": "Technology",
    "english_name": "Technology",
    "meta_title": "Technology News",
    "priority": 100
  }'
```
