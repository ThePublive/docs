title: "Create Category"

---

Creates a new category.

<ParamField path="publisher_id" type="string" required>
  Your Publisher ID
</ParamField>



<ParamField body="name" type="string" required>
  Category name
</ParamField>

<ParamField body="english_name" type="string" required>
  English name (used for permalink generation). **Cannot be changed after creation.**
</ParamField>

<ParamField body="slug" type="string">
  Custom slug (auto-generated from `english_name` if omitted). **Cannot be changed after creation.**
</ParamField>

<ParamField body="meta_title" type="string">
  SEO title
</ParamField>

<ParamField body="h1_tag" type="string">
  H1 heading tag
</ParamField>

<ParamField body="meta_description" type="string">
  SEO description
</ParamField>

<ParamField body="parent_category" type="integer">
  Parent category ID. **Immutable after creation.**
</ParamField>

<ParamField body="priority" type="integer">
  Priority level (1-1000)
</ParamField>

<ParamField body="content" type="string">
  Category description (HTML)
</ParamField>

<ParamField body="category_brand_color" type="string">
  Brand color (hex)
</ParamField>

<ParamField body="content_type" type="string">
  Content type filter (e.g., `Article`, `Web Story`). **Cannot be changed after creation.**
</ParamField>

#### Example Request

```bash
curl -X POST \
  'https://cms.thepublive.com/publisher/123/category/' \
  -H 'Authorization: Basic <BASE64_AUTH_TOKEN>' \
  -H 'Content-Type: application/json' \
  -d '{
    "name": "Technology",
    "english_name": "Technology",
    "meta_title": "Technology News",
    "priority": 100
  }'
```
