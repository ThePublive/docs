---
title: "Retrieve Form Schema"
description: Retrieve a single form schema by ID
api: "GET https://cms.thepublive.com/publisher/{publisher_id}/form/schema/{schema_id}/"
---

Returns the details of a specific form schema.

<ParamField path="publisher_id" type="string" required>
  Your Publisher ID
</ParamField>

<ParamField path="schema_id" type="string" required>
  Form schema ID (24-character hex string)
</ParamField>

<ParamField header="Authorization" type="string" required>
  `Basic <BASE64_AUTH_TOKEN>`
</ParamField>

#### Example Request

```bash
curl -X GET \
  'https://cms.thepublive.com/publisher/123/form/schema/6a3e5832f9ec326ec5277006/' \
  -H 'Authorization: Basic <BASE64_AUTH_TOKEN>'
```
