---
title: "Update Tag"
description: Update an existing tag
api: "PATCH https://cms.thepublive.com/publisher/{publisher_id}/tag/{id}/"
---

Updates an existing tag. Only send the fields you want to change (partial update).

<ParamField path="publisher_id" type="string" required>
  Your Publisher ID
</ParamField>

<ParamField path="id" type="integer" required>
  Tag ID
</ParamField>

<ParamField header="username" type="string" required>
  API Key
</ParamField>

<ParamField header="password" type="string" required>
  API Secret
</ParamField>

<ParamField body="name" type="string">
  Tag name
</ParamField>

<ParamField body="meta_title" type="string">
  SEO title
</ParamField>

<ParamField body="meta_description" type="string">
  SEO description
</ParamField>

<ParamField body="content" type="string">
  Tag description (HTML)
</ParamField>

#### Example Request

```bash
curl -X PATCH \
  'https://cms.thepublive.com/publisher/123/tag/500/' \
  -H 'username: YOUR_API_KEY' \
  -H 'password: YOUR_API_SECRET' \
  -H 'Content-Type: application/json' \
  -d '{"name": "Breaking News Updated"}'
```
