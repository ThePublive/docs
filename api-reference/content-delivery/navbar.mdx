---
title: "Navbar"
description: Fetch the navigation menu structure for your website
api: "GET https://cds.thepublive.com/publisher/{publisher_id}/navbar/"
---

Returns the navigation menu configuration with hierarchical support for nested menu items.

<ParamField path="publisher_id" type="string" required>
  Your Publisher ID
</ParamField>

<ParamField header="username" type="string" required>
  API Key
</ParamField>

<ParamField header="password" type="string" required>
  API Secret
</ParamField>

#### Response Fields

Each navbar item contains:

| Field | Type | Description |
| ----- | ---- | ----------- |
| `link` | string | URL path for the menu item |
| `name` | string | Display text |
| `children` | array/null | Nested child menu items (same structure) |
| `open_new_tab` | boolean | Whether to open link in a new tab |
