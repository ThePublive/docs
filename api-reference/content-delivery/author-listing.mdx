---
title: "Author Listing"
description: Fetch a list of all authors
api: "GET https://cds.thepublive.com/publisher/{publisher_id}/authors/"
---

Returns a paginated list of all authors with their profile information and social links.

<ParamField path="publisher_id" type="string" required>
  Your Publisher ID
</ParamField>

<ParamField header="username" type="string" required>
  API Key
</ParamField>

<ParamField header="password" type="string" required>
  API Secret
</ParamField>

<ParamField query="page" type="integer" default="1">
  Page number (max: 1000)
</ParamField>

<ParamField query="limit" type="integer" default="10">
  Items per page (max: 50)
</ParamField>
