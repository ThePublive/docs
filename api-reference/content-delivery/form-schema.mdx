---
title: "Form Schema (Reader)"
description: Fetch a form schema for rendering on the reader side
api: "GET https://cds.thepublive.com/publisher/{publisher_id}/form_schema/reader/{schema_id}/"
---

Returns a single form schema by ID, including field definitions, groups, and settings used to render the form on the reader side.

<ParamField path="publisher_id" type="string" required>
  Your Publisher ID
</ParamField>

<ParamField path="schema_id" type="string" required>
  Form schema ID (24-character hex string)
</ParamField>

<ParamField query="page_source" type="string">
  Optional context used by the serializer (if applicable)
</ParamField>
