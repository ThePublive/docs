---
title: "Category Details"
description: Fetch details of a single category by ID or slug
api: "GET https://cds.thepublive.com/publisher/{publisher_id}/category/{identifier}/"
---

Returns detailed information about a single category, including SEO metadata, Open Graph tags, and child categories.

<ParamField path="publisher_id" type="string" required>
  Your Publisher ID
</ParamField>

<ParamField path="identifier" type="string" required>
  Category ID or slug
</ParamField>

<ParamField header="username" type="string" required>
  API Key
</ParamField>

<ParamField header="password" type="string" required>
  API Secret
</ParamField>
